# job4j_cars

**Общее описание**

Этот учебный проект реализует УПРОЩЁННЫЙ сайт объявлений по продаже машин.

 ---

**Использованы технологии:**

Java 17,<br/>
maven 3.9.5,<br/>
Spring Boot 3.4,<br/>
Thymeleaf 3.4,<br/>
Bootstrap 4.1.3,<br/>
Liquibase 4.15.0,<br/>
PostgreSQL 16,<br/>
Junit 5.<br/>
Hibernate 5.6.11.Final<br/>

 ---

**Требования к окружению. (Список ПО, которое должно быть установлено для запуска проекта):**

Java 17, <br/>
Maven 3.8, <br/>
PostgreSQL 14<br/>

 ---

**Техническое задание**

1. Основная страница. таблица со всеми объявлениям машин на продажу.
2. На странице должна быть кнопка. добавить новое объявление.
3. Переходить на страницу добавления.
4. Должны быть категории машины, марка, тип кузова и тд.
5. Можно добавлять фото.
6. Объявление имеет статус продано или нет.
7. Должны существовать пользователи. Кто подал заявление. Только он может менять статус.

 ---

**Разделение прав:**

Просматривать объявления могут все.
Редактировать объявление может только тот, кто его опубликовал.

 ---

**Запуск проекта**(по шагам)**:**

1) Клонировать репозиторий.
   ```bash
   git clone https://github.com/alnesterenko/job4j_cars
   cd job4j_cars
   ```
2) Создать БД в PostgresSQL.
   ```bash
   CREATE DATABASE job4j_cars;
   ```
3) Внести необходимые изменения в БД, используя Liquibase.
   ```bash
   mvn liquibase:update
   ```
4) Запустить приложение.
   ```bash
   mvn spring-boot:run
   ```
