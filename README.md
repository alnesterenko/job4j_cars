# job4j_cinema
**Общее описание**

Этот учебный проект реализует УПРОЩЁННЫЙ сайт объявлений по продаже машин.

Использованы технологии:

Java 17,<br/>
maven,<br/>
Spring Boot 3.4,<br/>
Thymeleaf 3.4,<br/>
Bootstrap 4.1.3,<br/>
Liquibase 4.15.0,<br/>
Sql2o 1.6.0,<br/>
PostgreSQL 16,<br/>
Junit 5.<br/>

Hibernate ?!!<br/>


**Описание**
(в процессе будет уточняться)

На сайте должны быть объявления.
В объявлении должно быть:
 описание,
 марка машины,
 тип кузова,
 фото.

Объявление имеет статус продано или нет.
